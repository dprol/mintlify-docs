## Cómo usarlo

Genera preguntas relacionadas para profundizar en el problema y verificar tu comprensión.

## Estructura

- Preguntas que amplían el problema original
- Sistema de acordeón con respuestas expandibles
- Indicadores de dificultad para cada pregunta

## Navegación

Usa los botones de expansión para ver las respuestas detalladas.

## Atajo

Presiona <kbd>cmd/ctrl</kbd> + <kbd>Q</kbd> para generar nuevas preguntas.
