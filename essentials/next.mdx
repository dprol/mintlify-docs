## Cómo usarlo

Recibe sugerencias para el siguiente paso en tu código basadas en el contexto actual.

## Selección de código

1. Selecciona el fragmento de código relevante
2. Haz clic en "Próximo paso" en el menú de ayuda
3. Recibe sugerencias específicas para continuar

## Contexto inteligente

El asistente analiza:
- Variables y funciones definidas
- Estructuras de control utilizadas
- Estado actual del programa

## Formato de sugerencias

- Explicación del siguiente paso lógico
- Fragmento de código con sintaxis resaltada
- Razón detrás de la sugerencia

## Atajo de teclado

Usa <kbd>cmd/ctrl</kbd> + <kbd>N</kbd> con código seleccionado para obtener sugerencias rápidamente.
