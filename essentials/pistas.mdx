## Cómo usarlo

Obtén una lista de pistas iniciales para comenzar a resolver el problema.

## Obtener pistas

1. Selecciona el lenguaje de programación
2. Haz clic en "Pistas" en el menú de ayuda
3. Recibe tres pistas ordenadas por relevancia

## Formato

- Explicación clara y concisa
- Código de ejemplo cuando sea relevante
- Indicadores visuales de progreso

## Navegación

Usa las flechas o indicadores para moverte entre las pistas disponibles.

## Atajo rápido

Presiona <kbd>cmd/ctrl</kbd> + <kbd>H</kbd> para acceder directamente a las pistas.
