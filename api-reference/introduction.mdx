## Bienvenido

Buddy es un asistente de inteligencia artificial diseñado para integrarse con Visual Studio Code y ayudar a estudiantes principiantes en programación. Facilita el aprendizaje al proporcionar explicaciones claras, ejemplos prácticos y orientación para resolver problemas de programación.

## Características

Buddy ofrece una serie de funcionalidades clave que incluyen asistencia en la comprensión de conceptos, generación de ejemplos en pseudocódigo y diagramas de flujo, orientación en la resolución de problemas, y explicaciones paso a paso.
