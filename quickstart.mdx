## Instalación paso a paso

<AccordionGroup>
  <Accordion icon="browser" title="Instalar desde VS Code Marketplace">
    1. Abre Visual Studio Code
    2. Ve al panel de extensiones (Ctrl+Shift+X)
    3. Busca "BuddyAI"
    4. Haz clic en "Instalar"
  </Accordion>
  <Accordion icon="gear" title="Configuración inicial">
    1. Después de instalar, verás el icono de Buddy en la barra lateral
    2. Haz clic en el icono para abrir el panel de Buddy
    3. Selecciona tu lenguaje de programación preferido
  </Accordion>
</AccordionGroup>
